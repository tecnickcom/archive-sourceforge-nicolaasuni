1.1.000 (2005-01-14)
	- some new features were introduced:
	- shortcut# : new parameter that allows to specify a keyboard shortcut for each item (e.g.: SHIFT+A)
	- default_vmargin : default vertical space in pixels between buttons
	- default_hmargin : default horizontal space in pixels between buttons
	- default_padding : allows to specify a different padding value for left, right, top and bottom using the syntax L:R:T:B (e.g.: 2:3:10:4)
	- padding# : allows to specify a different padding value for left, right, top and bottom using the syntax L:R:T:B (e.g.: 2:3:10:4)
